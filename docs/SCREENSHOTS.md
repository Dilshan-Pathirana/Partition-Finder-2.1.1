# Screenshots

## How to Add Screenshots

To add screenshots to the README:

1. **Take screenshots** of:
   - Main GUI window
   - Analysis in progress
   - Results display
   - Configuration file selection

2. **Save images** in this `docs/` folder:
   - `gui-main.png` - Main interface
   - `gui-analysis.png` - Analysis running
   - `gui-results.png` - Results view
   - `gui-screenshot.png` - Overview screenshot

3. **Recommended format**: PNG, max width 800px

4. **Update README.md** - Images are already referenced, just place files here

**To take screenshots on Windows:**
- Press `Windows + Shift + S` for Snipping Tool
- Or use `PrtScn` and paste in Paint

**To take screenshots on macOS:**
- Press `Cmd + Shift + 4` and drag to select area
- Files save to Desktop

**To take screenshots on Linux:**
- Press `PrtScn` or use Screenshot tool
