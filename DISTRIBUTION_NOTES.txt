WHAT IS THIS?
-------------
This is a complete Python 3 modernization of PartitionFinder 2, originally
written for Python 2.7. This version works with Python 3.8 and higher.

IMPORTANT NOTES
---------------
1. This is an INDEPENDENT Python 3 port
2. NOT affiliated with the original PartitionFinder team
3. All scientific algorithms remain UNCHANGED
4. Only compatibility and modernization fixes applied

WHO SHOULD USE THIS?
-------------------
- Users who need PartitionFinder with modern Python (3.8+)
- Users on systems where Python 2.7 is no longer available
- Anyone wanting updated, maintained dependencies

QUICK START
-----------
1. Install Python 3.8 or higher (3.12 recommended)
2. Run: pip install -r requirements.txt
3. Test: py -3.12 PartitionFinder.py examples/nucleotide (Windows)
        or python3 PartitionFinder.py examples/nucleotide (Mac/Linux)

COMPLETE INSTRUCTIONS
---------------------
See HOW_TO_RUN.md for detailed step-by-step installation and usage guide.
